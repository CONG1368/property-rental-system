# 物业租赁综合管理系统

全栈物业租赁管理桌面应用，覆盖房源管理、租客管理、收租催缴、合同全生命周期、财务核算与报表分析。

## 技术栈

| 层 | 技术 |
|---|------|
| 前端框架 | Vue 3 Composition API + TypeScript |
| UI 组件库 | Element Plus |
| 图表 | ECharts 5 + vue-echarts |
| 状态管理 | Pinia |
| 后端框架 | Express + TypeScript |
| ORM | Sequelize 6（默认 SQLite，可选 MySQL） |
| 认证 | JWT（access 4h + refresh 7d） |
| 定时任务 | node-cron（账单生成/催缴/折旧/合同到期） |
| 实时通信 | WebSocket |
| 桌面打包 | Electron + electron-builder（NSIS 安装包） |

## 快速开始

### 环境要求

- Node.js ≥ 18
- npm ≥ 9

### 安装与启动

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务（前后端并行）
npm run dev
```

- 前端：http://localhost:5173
- 后端：http://localhost:3001
- 默认登录：`admin / admin123`

### 构建打包

```bash
npm run build          # 全局构建（前端 → 后端 → Electron）
npm run build:electron # 仅打包 Electron 安装包
```

### 数据库切换

默认使用 SQLite（零配置），编辑 `backend/.env` 切换为 MySQL：

```env
DB_DIALECT=mysql
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=yourpassword
DB_NAME=property_rental
```

## 项目结构

```
├── frontend/                  # Vue3 前端
│   └── src/
│       ├── api/               # API 请求模块
│       ├── components/        # 通用组件
│       ├── router/            # 路由（hash 模式）
│       ├── stores/            # Pinia 状态管理
│       └── views/             # 页面组件
│           ├── dashboard/     # 首页概览
│           ├── rent/          # 租赁管理
│           ├── finance/       # 财务管理
│           ├── contract/      # 合同管理
│           └── system/        # 系统设置
├── backend/                   # Express 后端（ESM）
│   └── src/
│       ├── models/            # Sequelize 模型（28个）
│       ├── routes/            # API 路由（24个模块）
│       ├── services/          # 业务服务层（19个服务）
│       ├── middleware/        # 中间件（认证/审计/校验/限流）
│       ├── jobs/              # 定时任务调度
│       └── websocket/         # WebSocket 实时推送
├── electron/                  # Electron 主进程
├── scripts/                   # 工具脚本
├── electron-builder.yml       # 打包配置
└── package.json               # 工程根配置
```

## 功能清单

### 租赁管理

| 功能 | 说明 |
|------|------|
| 房源管理 | 房源信息登记、状态跟踪、面积/户型/费用标准、批量导入 |
| 租客管理 | 租客信息、合同关联、信用评分 |
| 收租管理 | 账单生成、手动创建、收款记录、费用明细（租金/水费/电费/物业费/其他） |
| 智能催缴 | 三级催缴升级、催缴任务跟踪、逾期标记 |
| 门锁管理 | 智能门锁（远程开锁/密码管理/电量监控）、传统门锁（钥匙借还/挂失/作废流转）、操作日志 |
| 打印功能 | 合同/收据/账单/租客信息打印，支持系统原生打印和 PDF 导出，A4/热敏小票多规格 |
| 收租看板 | 收缴率、逾期率、收款渠道分布、收租趋势 |
| 收租日历 | 月度收租日历视图、批量状态一览 |

### 合同管理

| 功能 | 说明 |
|------|------|
| 合同列表 | 起草/审批/签署/执行/到期全状态管理 |
| 合同起草 | 表单式创建、费用配置（水/电/物业）、房源租客关联 |
| 合同审批 | 提交/驳回/签署流转、审批记录追溯 |
| 合同详情 | 完整信息展示、费用配置卡片、附件上传下载 |
| 合同看板 | 八列看板按状态分组 |
| 续约管理 | 到期前多周期预览（7/30/90/180/365天）、一键续约 |
| 到期日历 | 月度到期合同视图、多级到期统计 |
| 合同模板 | 模板管理 |
| 合规检查 | 合同合规性审查 |

### 财务管理

| 功能 | 说明 |
|------|------|
| 账套管理 | 多账套支持 |
| 会计科目 | 标准26科目体系 |
| 记账凭证 | 自动生成（收租→凭证）、手动录入 |
| 费用管理 | 维修/保洁/安保/绿化/办公/折旧等费用分类 |
| 税务管理 | 税务计算与导出 |
| 预算管理 | 预算编制/审核/执行跟踪 |
| 报表中心 | 收入报表、支出报表、利润分析 |
| 财务看板 | 关键财务指标一览 |

### 系统设置

| 功能 | 说明 |
|------|------|
| 用户管理 | 用户增删改、角色权限 |
| 字典管理 | 系统字典配置 |
| 打印设置 | 公司 Logo/签章/名称配置，实时预览 |

## 定时任务

| 任务 | 频率 | 说明 |
|------|------|------|
| 账单生成 | 每日 02:00 | 根据合同周期自动生成当期账单 |
| 催缴升级 | 每日 08:00 | 逾期账单自动升级催缴等级 |
| 合同到期标记 | 每日 07:00 | 到期合同状态自动更新 |
| 月度折旧 | 每月 1 日 02:00 | 固定资产月度折旧 |
